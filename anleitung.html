Start
Anleitung zur Steuerung

Durch Anklicken der unten aufgeführten Befehle kannst du diese schrittweise den gewünschten Funktionen zuweisen. Klicke anschließend auf „Start“, um die Befehlsfolge auszuführen, oder auf „Zurücksetzen“, um die Liste zu löschen und den Cursor auf die Startposition zurückzusetzen.

Bewegt den '➤' Cursor ein Feld nach vorne entsprechend der Richtung in der der Cursor zeigt.
Bewegt den '➤' Cursor um 90° nach links. Der Cursor bleibt auf dem selben Feld.
Bewegt den '➤' Cursor um 90° nach rechts. Der Cursor bleibt auf dem selben Feld.
Bewirkt das die Funktion F1 erneut ausgeführt wird.
Bewirkt das die Funktion F2 erneut ausgeführt wird.
Bewirkt das die Funktion F3 erneut ausgeführt wird.
